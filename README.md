# AppKebtang

แอปพลิเคชันบันทึกรายรับ-รายจ่าย (Expense Tracker)

---

## 🛡️ อัปเดตความปลอดภัยครั้งใหญ่ v2.0.1 (Security Hardening)

เวอร์ชันนี้มุ่งเน้นการยกระดับความปลอดภัยให้เป็นไปตามมาตรฐานความปลอดภัยทางไซเบอร์สากล เพื่อปกป้องข้อมูลส่วนตัวและรักษาความเป็นส่วนตัวของผู้ใช้สูงสุด

### 🛠️ สิ่งที่ได้รับการปรับปรุงตามมาตรฐานสากล:
- **Data Encryption & Hashing (มาตรฐาน NIST/OWASP)**:
    - ใช้ **Bcrypt (Work Factor 10)** ในการเข้ารหัสรหัสผ่าน ซึ่งเป็นมาตรฐานที่ทั่วโลกยอมรับ ป้องกันการถอดรหัสแม้ข้อมูลรั่วไหล
- **Secure Authentication (มาตรฐาน JWT/RFC 7519)**:
    - นำระบบ **JSON Web Token (JWT)** มาใช้ในการยืนยันตัวตน เพื่อให้การติดต่อสื่อสารระหว่างแอปและเซิร์ฟเวอร์มีความปลอดภัยและตรวจสอบสิทธิ์ได้แม่นยำ
- **Access Control (มาตรฐาน OWASP A01:2021)**:
    - ระบบ **IDOR Protection**: ตรวจสอบสิทธิ์ความเป็นเจ้าของข้อมูลทุกครั้งที่มีการเรียกใช้ API ป้องกันการเข้าถึงข้อมูลข้ามบัญชี
    - **Role-Based Access Control (RBAC)**: แยกสิทธิ์การใช้งานระหว่างผู้ใช้ทั่วไปและผู้ดูแลระบบอย่างเด็ดขาด
- **Infrastructure Security (มาตรฐาน OWASP A05:2021)**:
    - **Helmet.js**: เพิ่ม HTTP Security Headers เพื่อป้องกันการโจมตีแบบ XSS และ Clickjacking
    - **Rate Limiting**: จำกัดจำนวนการล็อกอินเพื่อป้องกันการสุ่มรหัสผ่าน (Brute-force Attacks)
- **Validation & Integrity**: บังคับความยาวรหัสผ่านขั้นต่ำและตรวจสอบความถูกต้องของข้อมูลก่อนบันทึกเสมอ

### ⚠️ การแก้ไขช่องโหว่จากเวอร์ชันก่อนหน้า:
จากการตรวจสอบประวัติ (Git Audit) เราได้กำจัดความเสี่ยงรุนแรงดังนี้:
1. **กำจัดการเก็บรหัสผ่านแบบข้อความปกติ**: เพื่อป้องกันการรั่วไหลของข้อมูลส่วนตัว
2. **ล็อคระบบหลังบ้าน (Admin Panel)**: ปิดกั้นการเข้าถึงข้อมูลผู้ใช้จากบุคคลภายนอกโดยสมบูรณ์
3. **ปิดช่องโหว่การเข้าถึงข้อมูลข้ามบัญชี**: มั่นใจได้ว่าข้อมูลของคุณจะเห็นได้เฉพาะคุณเท่านั้น
4. **ยกระดับความเข้มงวดของรหัสผ่านเริ่มต้น**: ยกเลิกรหัสผ่านที่คาดเดาง่ายและย้ายไปจัดการผ่านระบบลับ (Secrets Management)
