# AppKebtang

แอปพลิเคชันบันทึกรายรับ-รายจ่าย (Expense Tracker)

---

## 🛡️ อัปเดตความปลอดภัยครั้งใหญ่ v2.0.1 (Security Hardening)

เวอร์ชันนี้มุ่งเน้นการยกระดับความปลอดภัยให้เป็นไปตามมาตรฐานสากล (OWASP) เพื่อปกป้องข้อมูลส่วนตัวของผู้ใช้

### 🛠️ สิ่งที่ได้รับการปรับปรุง:
- **Password Hashing (bcrypt)**: เปลี่ยนการเก็บรหัสผ่านจากตัวอักษรปกติ (Plaintext) เป็นการเข้ารหัสที่ถอดรหัสไม่ได้ ปลอดภัย 100% แม้ฐานข้อมูลจะรั่วไหล
- **JWT Authentication**: นำระบบ Token มาใช้ในการยืนยันตัวตนแทนการส่งชื่อผู้ใช้ปกติ ทำให้การเชื่อมต่อระหว่างแอปและเซิร์ฟเวอร์ปลอดภัยขึ้น
- **IDOR Protection**: ระบบตรวจสอบสิทธิ์ทุกครั้งที่มีการเข้าถึงข้อมูล ป้องกันไม่ให้ผู้ใช้คนอื่นแอบดูข้อมูลของคุณผ่าน URL
- **Admin Security**: ล็อคหน้าแอดมินด้วยสิทธิ์ระดับสูงเท่านั้น และไม่แสดงรหัสผ่านของผู้ใช้ในหน้าแอดมินอีกต่อไป
- **Rate Limiting & Security Headers**: เพิ่มระบบป้องกันการสุ่มรหัสผ่าน (Brute-force) และเพิ่ม HTTP Security Headers (Helmet) เพื่อป้องกันการโจมตีทางไซเบอร์
- **Enhanced Validation**: บังคับรหัสผ่านอย่างน้อย 8 ตัวอักษร และตรวจสอบความซับซ้อนของข้อมูล

### ⚠️ คำเตือนความปลอดภัยจากเวอร์ชันก่อนหน้า (v2.0.0 และต่ำกว่า):
จากการตรวจสอบประวัติ (Git Audit) พบว่าเวอร์ชันก่อนหน้ามีความเสี่ยงรุนแรงดังนี้:
1. **รหัสผ่านไม่ถูกเข้ารหัส**: หากฐานข้อมูลรั่วไหล แฮกเกอร์จะเห็นรหัสผ่านทุกคนทันที
2. **หน้าแอดมินไม่มีการล็อค**: ใครที่รู้ URL สามารถเข้าถึงรายชื่อและรหัสผ่านของผู้ใช้ทั้งหมดได้
3. **ขาดระบบตรวจสอบสิทธิ์**: ผู้ใช้สามารถแอบดูข้อมูลของคนอื่นได้เพียงแค่ระบุชื่อผู้ใช้นั้นๆ ในคำสั่งเรียก API
4. **รหัสผ่าน Admin เริ่มต้นอ่อนแอ**: การใช้รหัสผ่าน "1234" ทำให้ง่ายต่อการถูกเดา

**แนะนำให้ผู้ใช้ทุกคนอัปเดตเป็น v2.0.1 ทันทีเพื่อความปลอดภัยของข้อมูล**
